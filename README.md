# BalloonDemo
很多App启动的时候会用到炫酷的开场动画。Android TV端也一样，每一个不同的模块，产品经理都可能设计了不同的开场动画。对于这些复杂的开场动画，最重要的是学会分解，只要分解得当，就会变成一个个普通的动画组合而成。这是一个曾经在项目中使用过的炫酷的开场动画。使用到了贝赛尔曲线、时差动画等知识点。
